# 📖 VSCode Extensions List  

This document lists and describes my favorite extensions for Visual Studio Code, organized by category.  

## 🔧 Development and Productivity  

- **[Better Comments](https://marketplace.visualstudio.com/items?itemName=aaron-bond.better-comments)** (`aaron-bond.better-comments@3.0.2`) - Improves the readability of code comments.  
- **[Path Intellisense](https://marketplace.visualstudio.com/items?itemName=christian-kohler.path-intellisense)** (`christian-kohler.path-intellisense@2.10.0`) - Auto-completes file and folder paths.  
- **[Material Icon Theme](https://marketplace.visualstudio.com/items?itemName=pkief.material-icon-theme)** (`pkief.material-icon-theme@5.20.0`) - Adds custom icons for files and folders.  
- **[Error Lens](https://marketplace.visualstudio.com/items?itemName=usernamehw.errorlens)** (`usernamehw.errorlens@3.25.0`) - Highlights errors and warnings directly in the code.  

## 🖥️ Web Development  

- **[Prettier - Code Formatter](https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode)** (`esbenp.prettier-vscode@11.0.0`) - Automatically formats code following best practices.  
- **[Auto Rename Tag](https://marketplace.visualstudio.com/items?itemName=formulahendry.auto-rename-tag)** (`formulahendry.auto-rename-tag@0.1.10`) - Automatically renames matching HTML and XML tags.  
- **[Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.liveserver)** (`ritwickdey.liveserver@5.7.9`) - Enables a local development server.  
- **[Thunder Client](https://marketplace.visualstudio.com/items?itemName=rangav.vscode-thunder-client)** (`rangav.vscode-thunder-client@2.34.3`) - HTTP client for testing REST APIs.  
- **[Live Preview](https://marketplace.visualstudio.com/items?itemName=ms-vscode.live-server)** (`ms-vscode.live-server@0.4.15`) - Previews web pages inside VSCode.  

## 🐘 PHP Development  

- **[Intelephense](https://marketplace.visualstudio.com/items?itemName=bmewburn.vscode-intelephense-client)** (`bmewburn.vscode-intelephense-client@1.14.4`) - Advanced IntelliSense for PHP.  
- **[PHP Debug](https://marketplace.visualstudio.com/items?itemName=xdebug.php-debug)** (`xdebug.php-debug@1.35.0`) - PHP debugging with Xdebug.  
- **[PHP Pack](https://marketplace.visualstudio.com/items?itemName=xdebug.php-pack)** (`xdebug.php-pack@1.0.3`) - A set of useful extensions for PHP development.  
- **[PHP Intellisense](https://marketplace.visualstudio.com/items?itemName=zobo.php-intellisense)** (`zobo.php-intellisense@1.3.3`) - Smart code suggestions for PHP.
