# 📖 Lista de Extensões para VSCode

Este documento lista e descreve as minhas extensões favoritas para o Visual Studio Code, organizadas por categoria.

## 🔧 Desenvolvimento e Produtividade

- **[Better Comments](https://marketplace.visualstudio.com/items?itemName=aaron-bond.better-comments)** (`aaron-bond.better-comments@3.0.2`) - Melhora a legibilidade dos comentários no código.
- **[Path Intellisense](https://marketplace.visualstudio.com/items?itemName=christian-kohler.path-intellisense)** (`christian-kohler.path-intellisense@2.10.0`) - Autocompleta caminhos de arquivos e pastas.
- **[Material Icon Theme](https://marketplace.visualstudio.com/items?itemName=pkief.material-icon-theme)** (`pkief.material-icon-theme@5.20.0`) - Adiciona ícones personalizados para arquivos e pastas.
- **[Error Lens](https://marketplace.visualstudio.com/items?itemName=usernamehw.errorlens)** (`usernamehw.errorlens@3.25.0`) - Destaca erros e avisos diretamente no código.

## 🖥️ Desenvolvimento Web

- **[Prettier - Code Formatter](https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode)** (`esbenp.prettier-vscode@11.0.0`) - Formata o código automaticamente seguindo boas práticas.
- **[Auto Rename Tag](https://marketplace.visualstudio.com/items?itemName=formulahendry.auto-rename-tag)** (`formulahendry.auto-rename-tag@0.1.10`) - Renomeia automaticamente tags HTML e XML.
- **[Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.liveserver)** (`ritwickdey.liveserver@5.7.9`) - Habilita um servidor local para desenvolvimento web.
- **[Thunder Client](https://marketplace.visualstudio.com/items?itemName=rangav.vscode-thunder-client)** (`rangav.vscode-thunder-client@2.34.3`) - Cliente HTTP para testar APIs REST.
- **[Live Preview](https://marketplace.visualstudio.com/items?itemName=ms-vscode.live-server)** (`ms-vscode.live-server@0.4.15`) - Pré-visualiza páginas web dentro do VSCode.

## 🐘 Desenvolvimento PHP

- **[Intelephense](https://marketplace.visualstudio.com/items?itemName=bmewburn.vscode-intelephense-client)** (`bmewburn.vscode-intelephense-client@1.14.4`) - IntelliSense avançado para PHP.
- **[PHP Debug](https://marketplace.visualstudio.com/items?itemName=xdebug.php-debug)** (`xdebug.php-debug@1.35.0`) - Depuração de código PHP com Xdebug.
- **[PHP Pack](https://marketplace.visualstudio.com/items?itemName=xdebug.php-pack)** (`xdebug.php-pack@1.0.3`) - Conjunto de extensões úteis para desenvolvimento PHP.
- **[PHP Intellisense](https://marketplace.visualstudio.com/items?itemName=zobo.php-intellisense)** (`zobo.php-intellisense@1.3.3`) - Sugestões inteligentes para PHP.

## 🏗️ Colaboração e Integração com GitHub

- **[GitLens](https://marketplace.visualstudio.com/items?itemName=eamodio.gitlens)** (`eamodio.gitlens@17.0.0`) - Melhora a integração do Git no VSCode.
- **[GitHub Copilot](https://marketplace.visualstudio.com/items?itemName=github.copilot)** (`github.copilot@1.293.0`) - Assistente de código baseado em IA.
- **[GitHub Copilot Chat](https://marketplace.visualstudio.com/items?itemName=github.copilot-chat)** (`github.copilot-chat@0.25.1`) - Chat baseado em IA para desenvolvimento.
- **[VS Live Share](https://marketplace.visualstudio.com/items?itemName=ms-vsliveshare.vsliveshare)** (`ms-vsliveshare.vsliveshare@1.0.5948`) - Colaboração em tempo real dentro do VSCode.

## ⚙️ Ferramentas Adicionais

- **[VS Code ESLint](https://marketplace.visualstudio.com/items?itemName=dbaeumer.vscode-eslint)** (`dbaeumer.vscode-eslint@3.0.10`) - Linter para JavaScript e TypeScript.
- **[Remote WSL](https://marketplace.visualstudio.com/items?itemName=ms-vscode-remote.remote-wsl)** (`ms-vscode-remote.remote-wsl@0.88.5`) - Permite usar o VSCode dentro do Windows Subsystem for Linux.
- **[Code Spell Checker](https://marketplace.visualstudio.com/items?itemName=streetsidesoftware.code-spell-checker)** (`streetsidesoftware.code-spell-checker@4.0.40`) - Verifica erros ortográficos em código e comentários.
- **[Color Highlight](https://marketplace.visualstudio.com/items?itemName=naumovs.color-highlight)** (`naumovs.color-highlight@2.8.0`) - Destaca cores em arquivos CSS e JS.
- **[Tabnine AI Autocomplete](https://marketplace.visualstudio.com/items?itemName=tabnine.tabnine-vscode)** (`tabnine.tabnine-vscode@3.250.0`) - Autocompletar de código baseado em IA.
- **[VSCode Preview Server](https://marketplace.visualstudio.com/items?itemName=yuichinukiyama.vscode-preview-server)** (`yuichinukiyama.vscode-preview-server@1.3.0`) - Servidor de visualização rápida para projetos web.

---
📌 **Dica:** Para instalar todas essas extensões de uma vez, use o comando no terminal do VSCode:

```sh
code --install-extension nome-da-extensão
```

🚀 **Mantenha seu VSCode otimizado e produtivo com essas extensões!**
