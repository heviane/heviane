# Code Editor

## [editorconfig](https://editorconfig.org)

Ferramenta para manter a consistência nas configurações do editor de código entre diferentes projetos e desenvolvedores. Ele permite definir e compartilhar configurações como estilo de indentação, quebra de linha e espaçamento. Com um arquivo ".editorconfig" no projeto, as preferências do editor são automaticamente aplicadas, promovendo consistência no estilo de código, independentemente do editor utilizado pela equipe. Isso ajuda a evitar discrepâncias e facilita a colaboração em projetos com diferentes membros e ambientes de desenvolvimento.
